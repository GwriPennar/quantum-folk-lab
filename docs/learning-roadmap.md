# Learning Roadmap

1. EXP-001: qubits, computational basis states, measurement, X/H/Z gates, superposition, repeated sampling, and Bell-state entanglement with local Qiskit/Aer simulation.
2. EXP-002: a small Max-Cut reference problem checked against brute force.
3. EXP-003: deterministic synthetic tune families.
4. EXP-004: a transparent two-family QUBO for graph partitioning.
5. EXP-005A: planned genuine local Qiskit QAOA, clearly separated from the current classical fallback sampler.
6. EXP-006: local noise sensitivity studies.
7. EXP-007: optional IBM hardware dry-runs and explicit QPU submission safeguards.
8. Later work: bounded public symbolic-music experiments after licence, provenance, privacy, and cultural-context review.

EXP-001 is local-only and requires no IBM account. It should be understood as ideal circuit education, not evidence of quantum advantage.
